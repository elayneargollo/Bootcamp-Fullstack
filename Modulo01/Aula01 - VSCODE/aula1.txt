INSTALAÇÃO DO VSCODE
================================

VISUAL CODE -> https://code.visualstudio.com/download

1 - ESCOLHA A VERSAO DO SISTEMA OPERACIONAL. 
2 - NA UNIDADE C, CRIE UMA PASTA PARA INSTALAÇÃO. 
3 - RECORTE DA PASTA DOWNLOAD O ARQUIVO BAIXAO PARA PASTA CRIADA.
4 - EXTRAIA O ARQUIVO COMPACTADO. 
5 - COPIE E COLE O EXECUTÁVEL DO VSCODE PARA ÁREA DE TRABALHO.

TORNA VSCODE PORTÁTIL
==================================

VISUAL CODE -> https://code.visualstudio.com/docs/editor/portable

1 - NA PASTA DE INSTALAÇÃO, CRIE UMA PASTA CHAMADA DATE.
2 - DENTRO DE DATE, CRIE UMA PASTA TMP.

BAIXAR FONTE FIRA CODE 
=====================================

FONTE -> https://github.com/tonsky/FiraCode

1 - EXTRAI O ARQUIVO.
2 - NO CASO DO WINDOWS, PROCURE A PASTA TTT.
3 - SELECIONE TODOS OS ARQUIVOS E COM O BOTÃO DIREITO CLICK EM INSTALAR.
